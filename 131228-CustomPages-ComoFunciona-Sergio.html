CaridadX "É como compras coletivas, só que para fazer o bem e ajudar pessoas..."

O CaridadX é uma plataforma especificamente voltada para a busca de ajuda por quem está precisando de ajuda.

Às vezes por motivo de doença, às vezes por motivo de acidente, ou qualquer que seja o motivo, acreditamos que ninguém, e nenhuma família, deve passar por momentos difíceis sozinhos

A seguir explicamos os passos para se criar uma causa para buscar ajuda.

1. Crie uma página da sua causa

Sabemos que a decisão de se criar uma página na Internet para buscar ajuda é uma decisão difícil. Por isso estamos aqui para ajudar. Não se intimide pela tecnologia. Acreditamos que a tecnologia também serve para ajudar pessoas, e é por esse motivo que criamos o CaridadX.

Para criar uma página para a sua causa, clique na opção CRIE SUA CAUSA, esse é o seu primeiro passo. Caso você não tenha cadastro no site ainda, uma tela solicitará o cadastro, que é grátis.

2. Se cadastrando no site

Caso você não tenha cadastro no site ainda, o próximo passo é realizar seu cadastro. Isso pode ser feito através do seu login do Facebook ou login do Google. Caso prefira, você pode criar seu próprio login no site clicando no botão "Registre-se Aqui".

3. Criando a sua causa

Após o cadastro (ou se você já fez o login), você está pronto para criar a sua causa. A criação das causas também é grátis.
